dweeter

A python module for messaging through the free dweet service. Dweet is a simple machine-to-machine (M2M) service from dweet.io.

Installation

pip install dweeter

Usage

>>> from dweeter import Dweeter
>>> dwtr = Dweeter("YOUR MAILBOX", "YOUR KEY")
